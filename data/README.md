# Dataset Description

This folder contains the datasets used for the Google Play Store App & User Review Analysis (EDA) project.

## Files

### `play_store_data.csv`
- Contains metadata for apps on the Google Play Store.
- **Columns include:**  
  App, Category, Rating, Reviews, Installs, Size, Type, Price, Content Rating, Genres, Last Updated, Current Ver, Android Ver

### `user_reviews.csv`
- Contains user review data for Google Play Store apps.
- **Columns include:**  
  App, Translated_Review, Sentiment, Sentiment_Polarity, Sentiment_Subjectivity

## Notes
- Large CSV files may not preview on GitHub due to size limits.
- Each dataset is used in the EDA notebook under the `notebooks/` folder.
